## How to Build a New Brain

See `templates/README.dev.md` for the full step-by-step guide. Quick checklist:

1. Create `brains/{slug}/` directory structure
2. Create Supabase tables from `templates/create-brain-tables.sql`
3. Fill `brain.json` with config (copy from scott-belsky, replace values)
4. Write `synthesis.md` (4-8 hours of manual research)
5. Ingest atoms into `{slug}_atoms` table
6. Run enrichment scripts (`enrich-voice.py`, `enrich-connections.py`)
7. Export pack (`export-brain.py`)
8. Register in `brains/index.json`
9. Audit with `scripts/audit-brains.py --brain {slug}`
